# 🤖 محلل المستندات بالذكاء الاصطناعي (AI Doc Analyzer)

تطبيق ويب ذكي واحترافي مصمم لتحميل ملفات السير الذاتية والمستندات بصيغة **PDF**، واستخراج النصوص منها تلقائياً، ثم معالجتها وتحليلها باستخدام نماذج الذكاء الاصطناعي وتوليد تقارير ملخصة واحترافية للمستخدم في واجهة تفاعلية حديثة بنمط الداكن الفاخر (SaaS Theme).

---

## 🚀 المميزات الرئيسية للمشروع

* **واجهة مستخدم عصرية (Modern SaaS Dashboard):** تصميم داكن فاخر متناسق بالكامل ومريح للعين مع تجربة مستخدم سلسة.
* **منطقة رفع تفاعلية:** دعم ميزة سحب وإفلات الملفات (Drag & Drop) مع مؤشرات تحميل (Loading Spinners) أثناء المعالجة.
* **استخراج نصوص دقيق:** معالجة ملفات الـ PDF وفك تشفير النصوص بداخلها بكفاءة عالية.
* **تحليل ذكي (AI-Powered):** الربط مع نماذج الذكاء الاصطناعي لاستنباط المهارات، الخبرات، والشهادات وتوليد خلاصة تنفيذية.
* **تنسيق تلقائي للنتائج:** دمج ميزة الـ Markdown Parser لترجمة ردود الذكاء الاصطناعي وعرض العناوين والنقاط الفرعية بشكل منسق وجذاب.

---

## 🛠️ التكنولوجيات المستخدمة (Tech Stack)

### 🔹 الخلفية البرمجية (Backend)
* **Python 3.10+**
* **FastAPI:** إطار عمل عالي الأداء وبناء الـ RESTful APIs.
* **Uvicorn:** سيرفر تشغيل تطبيقات ASGI بسرعة فائقة.
* **PdfPlumber / PyPDF:** لاستخراج النصوص من ملفات الـ PDF.

### 🔹 الواجهة الأمامية (Frontend)
* **HTML5 & CSS3:** بناء الهيكل وتصميم الـ UI بنمط الـ SaaS Theme.
* **JavaScript (Fetch API):** لإرسال واستقبال البيانات مع السيرفر خلف الكواليس دون الحاجة لإعادة تحميل الصفحة.
* **Marked.js (CDN):** لتحويل نصوص الـ Markdown القادمة من الذكاء الاصطناعي إلى عناصر HTML منسقة.
